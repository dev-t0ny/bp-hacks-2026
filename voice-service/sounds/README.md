# Fichiers audio

Déposez ici le fichier de son joué au démarrage d'une partie.

- **Fichier attendu par défaut :** `partie-commence.m4a`
- **Formats supportés :** mp3, wav, ogg, m4a (FFmpeg requis)

Vous pouvez aussi définir un chemin personnalisé via la variable d'environnement `SOUND_FILE_PATH` dans `voice-service/.env`.
